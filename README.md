       Challenges & Solutions
1. Managing Complex Component State
Challenge: State spread across many components became unpredictable.
Solution: Introduced controlled components, context, and improved data flow.
2. Styling Large Pages
Challenge: Styles became inconsistent as the project grew.
Solution: Adopted a shared style system using Tailwind utility groups and composition.
3. API Integration Errors
Challenge: Handling slow or failing endpoints.
Solution: Implemented graceful loading states, retry strategies, and TypeScript typing to avoid runtime failures.
4. Time Management
Challenge: Balancing deep learning with project deadlines.
Solution: Weekly planning using sprints, task breakdowns, and prioritization.

   Best Practices & Personal Takeaways
Build reusable components instead of copy-pasting UI.
Keep functions pure and predictable.
Document early and consistently.
Collaborate intentionally; frontend shines brighter with backend synergy.
Debugging is not a setback — it’s part of engineering.
Always test API assumptions (never trust the payload blindly).
Avoid over-engineering; simplicity scales best.
